 Student Management System (Java OOP MVP)

## Objective
This project is a simple Java console application that demonstrates the four main Object-Oriented Programming (OOP) concepts:
- Classes & Objects
- Encapsulation
- Inheritance
- Polymorphism

## Features
- Classes: Person, Student, Teacher
- Objects: Created in Main.java
- Encapsulation: Private fields with getters and setters (e.g., GPA validation)
- Inheritance: Student and Teacher extend Person
- Polymorphism:
  - Method overriding (displayInfo in Student and Teacher)
  - Method overloading (enrollCourse in Student)
  - Superclass reference pointing to subclass objects
